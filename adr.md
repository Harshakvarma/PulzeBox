Create a basic PWA React app that works in mobile applications which vibrates 4-4-4-4 rhythm breathing technique also known as box breathing built-in visual timer (with a clean graphics display) to help you set the pace for your breathwork.

4-4-4-4 means:
Inhale slowly while counting to 4.
Hold your breath for a count of 4.
Exhale steadily for a count of 4.
Hold your breath for a final count of 4

Must have Features:
- For Vibration use web-haptics/react library
- Use good animations of visulization
- Use audio signals with mute options for all 4 cycles

Options:
Newbie - 3 secs steps
Beginner - 4 sec steps (default option)
Advance - 5 sec increments
